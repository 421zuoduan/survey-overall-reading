# Top-Conference Supplement Agent

Search recent NeurIPS, ICML, ICLR, AAAI, and expandable venue proceedings using review-derived terms. Mark candidates with `top_conference_supplement`.

Down-rank and flag papers whose main novelty is prompt engineering. They may be retained as metadata-only frontier context, but should not enter the deep-reading queue unless they define an important benchmark or reveal a reusable mechanism.
